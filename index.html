Arcgis_Web AppBuilder for ArcGIS安装及说明

今天，我接触到了arcgis web端的AppBuilder，就整理一下关于AppBuilder的安装

一. 简介

1.描述

Web AppBuilder for ArcGIS 是一种直观的所见即所得式 (WYSIWYG) 应用程序，可用于构建 2D 和 3D web 应用程序，而无需编写一行代码。它所包括的强大工具可用来配置功能完备的 HTML 应用程序。添加地图和工具时，您可以在应用程序中看到这些地图和工具并立即使用。

2.功能

  • 创建能在所有设备上运行的 HTML/JavaScript 应用程序
  • 使用即用型微件构建所需应用程序
  • 使用可配置的主题自定义应用程序的外观
  • 在线托管应用程序或在自己的服务器上运行应用程序
  • 创建自定义应用程序模板
上述是官方描述的，总之就是
功能强，操作简单，不需要代码，好上手 +_+

现在我把安装方式和视频教程的网盘链接放上 

https://pan.baidu.com/s/1Kz-Wlp58KVP-NC8hN5V01g 提取码：gq8k

二.安装方法

这是官网安装的链接，讲述的很全，我这里简单叙述一下
https://developers.arcgis.com/web-appbuilder/guide/getstarted.htm#ESRI_SECTION1_F2CE6567CB854E4AB43F08EB64B3D7AE

（1）解压“arcgis-web-appbuilder-2.13.zip”文件

（2）双击其中的”startup.bat”文件，电脑会自动打开命令提示窗口，同时会在默认浏览器中启动Web AppBuilder服务器，其中带有http：//[您的计算机名称]：3344/webappBuilder

小提示：这里据说需要Node.js 服务器才可以，但是我直接成功了，可能之前装过，如果打不开可以找一个教程

（3）若成功的话，在浏览器中可以发现有两个框需要输入内容

  • 指定门户url
  • 应用 ID

    上述是意味着需要连接到我们arcgis online中，也就是需要有一个arcgis online的账户，可以注册一个。

    （4）对arcgis online设置应用程序

(5)返回一开始网页，指定门户可以填自己online地址

如https://xxxx.maps.arcgis.com

(6)设置应用ID

A.arcgis online中打开我的内容 ,选择我们建立的构建器，点击右方的“…” ,选择“查看项目详细信息”

B.单击”设置”选项卡。向下滚动到”应用注册”部分，然后单击”注册”。
对于ArcGIS 的门户，单击应用注册部分中的注册。

在"注册"对话框上
应用类型————请保留默认值
重定向 URI————请输入http：//[您的计算机名称]或https：//[您的计算机名称]
  请注意，具有 HTTP 和 HTTPS 协议的 URI 被区分为两个不同的 URI。
C.单击注册，在应用注册下复制应用 ID，粘贴到Web AppBuilder的文本框中，然后单击"继续"

再次说明，具体可以参考以下链接，包括教程

https://developers.arcgis.com/web-appbuilder/guide/build-your-first-app.htm

ArcGIS制图及出图小技巧—以土地利用图为例

一. 概述

这里讲述关于ArcGIS制图出土小技巧，主要包括以下方面：

  • Arcgis取色

    • 颜色应用

    • 数据框调整

    • 出图

二. 操作

1. ArcGIS取色

如过发现自己的地图颜色老是配的不好可以采用其他已经制作好的地图颜色，可以通过 外部取色器 或者 arcgis内部取色器 进行取材。

（1）.外部取色器：好多大佬为了作图方便，制作的颜色抓取工具，这些在网上可以搜到安装包，可以直接查看颜色的值。下图为我的一个颜色提取工具。下面是网盘链接。

链接：https://pan.baidu.com/s/1xKyPLq7yqVFJ5I-wueqsrQ 提取码：iv6w

（2）.ArcGIS内部取色器：属于arcgis内部的功能，操作如下

A.打开ARCGIS
B.拖入具有色彩的图片
如同拖入shp格式的方式将图片拖入arcgis
C.打开取色器
第一次操作需要在菜单栏【自定义】—【自定义模式】—【命令】—搜索【取色器】— 将取色器拖到菜单栏上

D.选取颜色，并命名
如下图选中湖泊的颜色并备注，单击确定

E.查看是否设置成功
设置好后可以打开【自定义】—【样式管理器】—【颜色】可以看到我们设置好的颜色

2.颜色应用

(1)修改颜色

当选取颜色后，对图层符号系统设置颜色时，选择填充就可以看到我添加的样式被放置在颜色表前面的位置，选择即可。

（2）批量修改

若制作多年分布对比图，若单独制作的话颜色回事随机的，可以做好一年的然后另外两个图层采用导入的方式设置

3. 数据框调整

为了多年数据进行对比，可以及新建多个数据框

（1）适应纸张的调整

在ArcGIS出图时，需要设置纸张大小，将显示的数据框放到纸张框中，导出纸张框内部的图。

（2）数据框的排版

选中建好数据框，集体右击，分布，将数据框设置成相同大小，选中单独数据框-设置全图显示 通过【对齐】—【顶端对齐】 【分布】—【水平分布】设置，
排版时也可以打开标尺进行设置，将数据框对齐。

(3) 图例设置

如下图，选择【插入】-【图例】，单击下一步直至完成，可以看到插入的图例，但是图例有一些不需要的部分，可以打开图例的数据框中图层的属性将不需要的标注删除即可，同时添加标注可以

右击图例，设置项目，将列设置为”n”列，”n”自己设置。可以看到如下效果

基于arcgis时间动态图的制作

一.简述

  • 地理信息系统是在计算机硬件、软件得支持下，对整个或部分地球标称空间中得有关地理分布数据进行采集、储存、管理、运算、分析、显示和描述得技术系统
  • 这里以中国肺炎疫情时间动态图为例

    二. 操作说明

    1.打开Arcgis map，并新建地图

    2.导入中国省级地图数据

  • 注:打开后如属性表有乱码问题，则说明编码方式不正确，在源数据得文件里写入一个cpg文件，在内表明编码方式

    3.编辑源数据属性表：删除找到的源数据得无用字段，保留必须字段（如fid、地区名称等）

    4.编辑自己得属性表：以保留的字段为关键字段在excel编辑属性表，编辑好后要以xsl格式保存，否则arcgis会不可识别

    5.导入属性表：点击  添加数据  找到设置xsl表的位置，点击添加

  • 注：要预先在arcgis中关联文件夹（目录-文件夹连接-选择要关联的文件夹）

    6. 连接字段

    (1). 选择导入的地图右击————“连接和关联”————”连接“
    (2). 选择连接的表以及基础字段（指基于什么字段连接，如各省名称）
    (3). 点击“验证连接”，连接成功后点击“确定”即可

  • 注：如为防止之后不出现问题最好将连接后的地图进行一次导出（右击地图——点击“数据”——“导出数据”——选择“数据框”——选择导出的路径——新建名称——点击“确定”，之后的地图可以采用导出的新地图进行操作）

    7.字段转置

  • “工具箱”———“数据管理工具”————“字段”————“字段转置”

  • 选择要转置的字段以及存储路径，编辑字段明称以及字段值，保留保留的字段后确认

  • 注：这里可能我学艺不精，导入的字段类型会变为以F4，F5,F6以此类推的Fn，我认为是由于zaiexcel表格中数据格式不同导致，但是我没有解决的办法，查了许多资料没卵用，故在转置后打开属性表进行字段替换

    8.属性表再次编辑

  • 这个问题主要还是数据类型问题，我的式疫情时间变化图，故字段为时间（也就式上文说的Fn的问题，将Fn替换为对应的日期后2020/1/20，但格式认为字符串，需进行变换）

        注：格式变换方法
         (1) 字符串转日期
         先设置标准时间格式，如上文设置好格式2020/1/20（必须） 打开“工具箱”——“数据管理工具”——“字段”——“抓换时间字段”——后按表中进行转换
         (1) 字段转换——数值
         转换数值的话可利用字段计算器进行转换

    9.打开时间图层

  • 右击“图层”———打开“属性”———选择“时间”———勾选“在此图层打开时间”———然后根据自己需求设置
  • 在窗口打开“时间对话框”

    10.再对属性进行设置，做出时间动态图

    右击“图层”———打开“属性”———选择“符号系统”———再进行自己喜欢的显示设置

windows系统的云服务器发布网页

一.购买云服务器

  很多平台可以购买云服务器（如阿里云、腾讯云、华为云、百度云}

1. 阿里云平台（https://www.aliyun.com/）

2. 腾讯云平台（https://cloud.tencent.com/）

二.云服务器设置

我这里用的是阿里云服务器

  1. 在电脑左下角搜索框输入“远程桌面连接”，计算机输入自己云服务器提供的公网ip，用户名输入“Administrator”
  2. 输入云服务器设置的密码，即可进入服务器

    三.设置IIS

  3. 打开控制面板
  4. 选择应用或关闭windows功能
  5. 开启IIS共能（Internet Information Services）
  6. 打开IIS
    可在左下角搜索框直接搜索IIS，为了以后方便也可以直接在桌面设置快捷方式，或者固定至开始屏幕

    四.新建站点

  7. 将建好的文件夹上传至服务器（可直接复制，在服务器粘贴）
  8. 打开IIS，新建一个站点。右击“网站”标签，选择添加网站，按下图进行设置
  9. 设置完毕后，点击浏览，即可在本地浏览器看自己做的网页

    五.设置防火墙

  10. 打开 控制面板——>系统和安全——>windows defender防火墙，然后点开左边高级设置。如图
  11. 配置入站规则，点击左边入站规则
  12. 单击 新建规则，规则类型选择端口
  13. 规则应用选择”TCP”,选择特定本地端口，端口输入80。继续点击下一步
  14. 选择允许连接，点击下一步
  15. 默认都选择，不需要修改，点击下一步
  16. 名称随便填一个，点击完成

    六.设置安全组

  17. 打开自已的控制台
  18. 打开自己购买的服务器实例
  19. 点击左侧本实例安全组
  20. 点击配置
  21. 选择添加安全组规则
  22. 端口范围填入“80/8081” 授权对象填入0.0.0.0/0

    六.在浏览器输入自己的服务器公网域名就可以使用了！！！
