hexo主题设置———tags与categories的设置

注： 这里主要针对对于landscape主题的设置和更改

1. 设置主题（landscape）

一般安装hexo，默认文件为landscape，但一些新手对于某些功能不是很明确

  • 站点配置文件：_config.yml
  • 主题配置文件：themes/landscape/_config.yml

    2. 标签设置（tags）

    (1) 在站点配置文件中查看是否有tag_dir: tags
    (2) 在主题配置文件中查看是否有tags: /tags
    (3)编辑站点的source/tags/index.md

    3.分类设置（categories）

    (1) 在站点配置文件中查看是否有category_dir: categoriess
    (2) 在主题配置文件中查看是否有categories: /categories
    (3)编辑站点的source/categories/index.md

一. hexo

  Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
本博客就是借由Hexo搭建而成。

二.博客搭建后文件夹简单说明

（1）_config.yml 网站的配置信息

（2）package.json 应用程序的信息，可删除

（3）sacffolds 模板文件夹。新建的文章会根据其建立文件。（可以先建好格式，之后统一以其为模板）

（4） source 资源文件夹存放用户资源的地方

（5）themes 主题文件夹

hexo的常用命令

（1） hexo new <标题> 新建文章

（2）hexo generate 生成静态文件

（3）hexo publish 发布草稿

（4）hexo server 启动服务器。默认情况下

访问网址为： http://localhost:4000/

（5）hexo deploy 部署网站

（6）hexo clean 清除缓存文件

  更多操作可以点这里查询

服务器

windows系统的云服务器发布网页

一.购买云服务器

  很多平台可以购买云服务器（如阿里云、腾讯云、华为云、百度云}

1. 阿里云平台（https://www.aliyun.com/）

2. 腾讯云平台（https://cloud.tencent.com/）

二.云服务器设置

我这里用的是阿里云服务器

  1. 在电脑左下角搜索框输入“远程桌面连接”，计算机输入自己云服务器提供的公网ip，用户名输入“Administrator”
  2. 输入云服务器设置的密码，即可进入服务器

    三.设置IIS

  3. 打开控制面板
  4. 选择应用或关闭windows功能
  5. 开启IIS共能（Internet Information Services）
  6. 打开IIS
    可在左下角搜索框直接搜索IIS，为了以后方便也可以直接在桌面设置快捷方式，或者固定至开始屏幕

    四.新建站点

  7. 将建好的文件夹上传至服务器（可直接复制，在服务器粘贴）
  8. 打开IIS，新建一个站点。右击“网站”标签，选择添加网站，按下图进行设置
  9. 设置完毕后，点击浏览，即可在本地浏览器看自己做的网页

    五.设置防火墙

  10. 打开 控制面板——>系统和安全——>windows defender防火墙，然后点开左边高级设置。如图
  11. 配置入站规则，点击左边入站规则
  12. 单击 新建规则，规则类型选择端口
  13. 规则应用选择”TCP”,选择特定本地端口，端口输入80。继续点击下一步
  14. 选择允许连接，点击下一步
  15. 默认都选择，不需要修改，点击下一步
  16. 名称随便填一个，点击完成

    六.设置安全组

  17. 打开自已的控制台
  18. 打开自己购买的服务器实例
  19. 点击左侧本实例安全组
  20. 点击配置
  21. 选择添加安全组规则
  22. 端口范围填入“80/8081” 授权对象填入0.0.0.0/0

    六.在浏览器输入自己的服务器公网域名就可以使用了！！！
